# NILEFLIX

**NILEFLIX** is a Netflix-inspired streaming platform UI created as a university project. The website showcases a modern movie browsing experience with a responsive design, featured hero section, and detailed movie pages. The project focuses on front-end development using clean HTML and CSS while recreating the look and feel of a popular streaming service.

## Features

- 🎬 Modern Netflix-inspired interface
- 📺 Featured hero banner
- 🎞️ Movie detail page
- 📱 Responsive layout
- ✨ Clean and simple UI

## Technologies Used

- HTML
- CSS
